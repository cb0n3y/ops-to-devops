# Install vagrant

With vagrant installed you are able to create, start, stop and destroy with simple commands.

## Prerequisites

You will need:

- the [Vagrant CLI](https://developer.hashicorp.com/vagrant/tutorials/get-started/install) installed locally on your machine
- [VirtualBox_7.1.6](https://www.virtualbox.org/wiki/Downloads) installed as your virtualization provider

## Where to start with?

Well, you can read the documentation and try it on your own or you just can go to [Learn Vagrant get started] (https://github.com/hashicorp-education/learn-vagrant-get-started) and see some examples how it works.

## Commands

```vagrant up``` will start tzhe vms declared on the file. If you want to start only one vm, just use the same command with the vm name: ```vagrant up <vm-name>```.
